<h3>源码简介：</h3>
<p>使用方法：运行程序后自动锁定置顶F1，取消置顶F2，若要修改热键，重新选择热键后单击“确定”按钮即可。
鼠标光标移到目标窗口上按下置顶热键（如F1）即可自动置顶，若要取消置顶，鼠标光标移到目标窗口上按下取消热键（如F2）即可取消置顶。<p>
